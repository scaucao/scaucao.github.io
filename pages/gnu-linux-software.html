GNU/Linux and Software

[ MatemaTeX ]
I created this project in January 2013 to facilitate and complement the work of mathematical students and teachers. The main idea of this web-site is to save in the clipboard (with just a click) the LaTeX code of mathematical excercises.


[ Osorno Linux ]
This is a non-profit group created in May 2010 dedicated to diffusing, teaching and learning about GNU/Linux, open-source software (OSS) and related projects. Members: L. Gomez, P. Zúñiga, and me.
